# Vue2 到 Vue3 迁移 Skill 文件检查任务

## 任务目标
检查 `agents/vue2-to-vue3-upgrade/skills/` 目录下的 skill 文件是否正确地从 `agents/vue2-to-vue3-upgrade/breaking-changes/` 目录下的文档转换而来。

## 文件列表
共有 31 个文件需要检查（排除 migration-skills-template.md）。

## 检查维度

### 1. 规则完整性检查
- [ ] 是否遗漏了原始文档中的任何迁移规则
- [ ] 原始文档中的"迁移策略"、"2.x 语法"、"3.x 语法"等章节是否都被提取并转换为规则

### 2. 内容准确性检查
- [ ] Vue2 写法是否准确对应原文档的 2.x 语法
- [ ] Vue3 写法是否准确对应原文档的 3.x 语法
- [ ] 代码示例是否完整且正确
- [ ] 是否遗漏了重要的代码示例

### 3. 格式规范检查
- [ ] frontmatter 是否包含 name 和 description
- [ ] 是否包含"迁移规则概览"章节
- [ ] 每个规则是否包含"Vue2: xxx"、"Vue3: xxx"对比
- [ ] 每个规则是否包含"代码示例"章节
- [ ] 代码块语言标识是否正确

### 4. 特殊情况检查
- [ ] 原始文档包含多个相关变更时，是否都拆分为独立规则
- [ ] 原始文档包含表格、列表等格式内容时，是否正确提取
- [ ] 是否正确处理了"迁移构建开关"、"参考"等附加信息

## 检查方法

对于每个文件（如 `v-model.md`）：
1. 读取 `breaking-changes/v-model.md` 原始文档
2. 读取 `skills/v-model.md` 转换后的 skill 文件
3. 逐条对比分析，记录：
   - 遗漏的规则
   - 不准确的 Vue2/Vue3 对比
   - 缺失或错误的代码示例
   - 格式问题

## 输出格式

为每个文件输出检查报告：

```
### 文件名: xxx.md

✅ 通过项
- [具体通过的内容]

❌ 发现问题
- **规则遗漏**: [具体遗漏的规则描述]
- **内容错误**: [具体错误描述，附原始内容 vs 转换内容]
- **格式问题**: [具体格式问题]

📝 修正建议
[针对每个问题的具体修正建议]

---

[总体评估]
```

请逐个文件进行检查，输出完整的检查报告。如发现问题，请提供具体的原文档内容和转换后内容的对比。
