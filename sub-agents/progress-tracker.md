# 进度追踪器

## 职责

显示升级进度和结果摘要。

## TODO

- [ ] 读取项目根目录的 `.vue3-upgrade-tasks.json` 文件
- [ ] 解析任务文件，提取统计信息
- [ ] 生成进度报告，包含:
  - 总体进度 (已完成 / 总文件数 百分比)
  - 文件状态统计 (已完成 / 待处理 / 失败)
  - TODO 标记统计 (总数)
  - 问题统计 (高 / 中 / 低优先级)
  - 失败文件列表 (如有)
  - 下一步操作建议
- [ ] 按照指定格式输出进度报告

## 输出格式

```
=== Vue2 到 Vue3 升级进展 ===

📊 总体进度: [XX]%
   已完成: X / Y 文件

📋 文件状态:
   ✅ 已完成: X
   ⏳ 待处理: Y
   ❌ 失败: Z

📝 变更统计:
   - TODO 标记总数: N
   - 高优先级问题: A
   - 中优先级问题: B
   - 低优先级问题: C

❌ 失败文件:
   - src/components/Failed.vue: 语法解析失败

🔄 下一步:
   继续迁移下一个文件或查看失败文件
```
